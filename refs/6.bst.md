# Review of Complexity of Algorithms: Big-O Notation

Analyzing an algorithm helps predict required resources:

- Memory
- Communication bandwidth
- Computational time (usually most important)

The running time usually depends on **input size N**.

Examples:

- Sorting → number of elements
- Matrix multiplication → total matrix elements

Big-O notation:

```
O(f(N))
```

It represents the **asymptotic upper bound** of algorithm growth.

---

# Typical Growth Rates

Running time:

```
αf(N) + β
```

Common complexity classes:

| Function | Name |
|---|---|
| c | constant |
| log N | logarithmic |
| log² N | log-squared |
| N | linear |
| N log N | linearithmic |
| N² | quadratic |
| N³ | cubic |
| 2ⁿ | exponential |

# Recursive Definition of Trees

A tree **T** is:

Base case:

```
T is empty
```

Recursive definition:

```
T = root r + subtrees T1, T2, ... Tk
```

---

# Tree Terminology

- **Root**: node with no parent
- **Parent / Child**
- **Leaf**: node with no children
- **Siblings**: nodes sharing same parent

Path:

```
{n1, n2, ..., nk}
```

where each node is the parent of the next.

---

# Additional Tree Terminology

Path length:

```
number of edges
```

Depth of node:

```
distance from root
```

Height of node:

```
longest path from node to leaf
```

Height of tree:

```
height(root)
```

Ancestor / Descendant:

If path exists:

```
n1 → n2
```

then

```
n1 ancestor
n2 descendant
```

# Part II  
## Binary Tree

A **binary tree** is a tree where each node has **at most two children**.

Properties:

- Average height ≈ **O(log N)**
- Worst-case height = **N−1**

---

# Binary Tree Implementation

```cpp
template <class T>
class BTnode
{
public:
    BTnode(const T& x, BTnode* L=nullptr, BTnode* R=nullptr)
        : data(x), left(L), right(R) {}

    ~BTnode()
    {
        delete left;
        delete right;
        cout<<"delete the node with data = "<<data<<endl;
    }

    const T& get_data() const { return data; }
    BTnode* get_left() const { return left; }
    BTnode* get_right() const { return right; }

private:
    T data;
    BTnode* left;
    BTnode* right;
};
```

---

# Binary Tree Traversal: Inorder

Traversal order:

```
left subtree → node → right subtree
```

```cpp
template <class T>
void btree_inorder(BTnode<T>* root,
                   void (*action)(BTnode<T>* r))
{
    if(root)
    {
        btree_inorder(root->get_left(), action);
        action(root);
        btree_inorder(root->get_right(), action);
    }
}
```

---

# Binary Tree Traversal: Preorder

Traversal order:

```
node → left subtree → right subtree
```

```cpp
template <class T>
void btree_preorder(BTnode<T>* root,
                    void (*action)(BTnode<T>* r))
{
    if(root)
    {
        action(root);
        btree_preorder(root->get_left(), action);
        btree_preorder(root->get_right(), action);
    }
}
```

---

# Binary Tree Traversal: Postorder

Traversal order:

```
left subtree → right subtree → node
```

```cpp
template <class T>
void btree_postorder(BTnode<T>* root,
                     void (*action)(BTnode<T>* r))
{
    if(root)
    {
        btree_postorder(root->get_left(), action);
        btree_postorder(root->get_right(), action);
        action(root);
    }
}
```

---

# Binary Tree Example

```cpp
template <typename T>
void print(BTnode<T>* root)
{
    cout << root->get_data() << endl;
}

int main()
{
    BTnode<int>* node5 = new BTnode<int>(5);
    BTnode<int>* node9 = new BTnode<int>(9);
    BTnode<int>* node8 = new BTnode<int>(8,node5,node9);

    BTnode<int>* node12 = new BTnode<int>(12);
    BTnode<int>* node17 = new BTnode<int>(17);
    BTnode<int>* node15 = new BTnode<int>(15,node12,node17);

    BTnode<int>* root = new BTnode<int>(10,node8,node15);

    btree_inorder(root,print);
    btree_preorder(root,print);
    btree_postorder(root,print);

    delete root;
}
```

# Part III  
## Binary Search Tree (BST)

BST property:

For node **x**

```
left subtree values < x
right subtree values > x
```

BST supports:

- Search
- Insert
- Delete

---

# BST Properties

Different insertion orders can produce different BSTs.

Depth:

```
average = O(log N)
worst case = O(N)
```

---

# BST Implementation

```cpp
template <typename T>
class BST
{
private:

    struct BSTnode
    {
        T value;
        BST left;
        BST right;

        BSTnode(const T& x):value(x){}
        BSTnode(const BSTnode&) = default;

        ~BSTnode(){ cout<<"delete: "<<value<<endl; }
    };

    BSTnode* root=nullptr;

public:

    BST()=default;

    ~BST()
    {
        delete root;
    }

    bool is_empty() const
    {
        return root==nullptr;
    }

    bool contains(const T&) const;

    void print(int depth=0) const;

    const T& find_max() const;
    const T& find_min() const;

    void insert(const T&);
    void remove(const T&);
};
```

---

# BST Search

```cpp
template <typename T>
bool BST<T>::contains(const T& x) const
{
    if(is_empty())
        return false;

    if(root->value == x)
        return true;

    else if(x < root->value)
        return root->left.contains(x);

    else
        return root->right.contains(x);
}
```

Time complexity:

```
O(height of BST)
```

---

# BST Print (Rotated)

```cpp
template <typename T>
void BST<T>::print(int depth) const
{
    if(is_empty())
        return;

    root->right.print(depth+1);

    for(int j=0;j<depth;j++)
        cout<<"\t";

    cout<<root->value<<endl;

    root->left.print(depth+1);
}
```

---

# Find Minimum

The minimum value is the **leftmost node**.

```cpp
template <typename T>
const T& BST<T>::find_min() const
{
    const BSTnode* node=root;

    while(!node->left.is_empty())
        node=node->left.root;

    return node->value;
}
```

---

# Find Maximum

The maximum value is the **rightmost node**.

```cpp
template <typename T>
const T& BST<T>::find_max() const
{
    const BSTnode* node=root;

    while(!node->right.is_empty())
        node=node->right.root;

    return node->value;
}
```

---

# BST Insertion

```cpp
template <typename T>
void BST<T>::insert(const T& x)
{
    if(is_empty())
        root=new BSTnode(x);

    else if(x < root->value)
        root->left.insert(x);

    else if(x > root->value)
        root->right.insert(x);

    else
        ;
}
```

Time complexity:

```
O(height)
```

---

# BST Deletion Cases

Case 1: **Leaf node**

```
delete directly
```

Case 2: **One child**

```
replace node with child
```

Case 3: **Two children**

Replace node with:

```
max(left subtree)
or
min(right subtree)
```

---

# BST Remove Implementation

```cpp
template <typename T>
void BST<T>::remove(const T& x)
{
    if(is_empty())
        return;

    if(x < root->value)
        root->left.remove(x);

    else if(x > root->value)
        root->right.remove(x);

    else if(root->left.root && root->right.root)
    {
        root->value = root->right.find_min();
        root->right.remove(root->value);
    }
    else
    {
        BSTnode* deleting_node = root;

        root = (root->left.is_empty())
            ? root->right.root
            : root->left.root;

        deleting_node->left.root = nullptr;
        deleting_node->right.root = nullptr;

        delete deleting_node;
    }
}
```
