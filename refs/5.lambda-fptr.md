# Lambda Expressions (Lambdas)

Syntax:

```
[ <capture-list> ] ( <parameter-list> ) mutable -> <return-type> { <body> }
```

Properties:

- Lambdas are **anonymous functions** (functions without names).
- Usually defined **inside other functions**.
- Can capture variables from surrounding scope.

### Capture List

Common capture forms:

```
[=]    capture all local variables by value
[&]    capture all local variables by reference
[x,y]  capture only specific variables
```

Notes:

- Global variables **do not need to be captured**.
- Capturing global variables is **an error**.

### Return Type

Return type:

- `void` by default if there is **no return statement**
- automatically **inferred if return exists**
- may be explicitly specified with `->`

---

# Example: Simple Lambdas (No Capture)

```cpp
#include <iostream>
using namespace std;

int main()
{
    int range[] = {2, 5, 7, 10};

    // Lambda computing square
    for (int v : range)
        cout << [](int k){ return k*k; }(v) << endl;

    // Lambda doubling numbers
    for (int& v : range)
        [](int& k){ return k*=2; }(v);

    for (int v : range)
        cout << v << "\t";

    cout << endl;

    // Lambda computing max
    int x[3][2] = {{3,6},{9,5},{7,1}};
    for (int k=0;k<sizeof(x)/sizeof(x[0]);++k)
        cout << [](int a,int b){ return (a>b)?a:b; }
        (x[k][0],x[k][1]) << endl;

    return 0;
}
```

# Example: Lambdas with Captures

```cpp
#include <iostream>
using namespace std;

int main()
{
    int sum=0, a=1, b=2, c=3;

    // quadratic polynomial
    for(int k=0;k<4;++k)
        cout << [=](int x){ return a*x*x + b*x + c; }(k) << endl;

    cout << "a="<<a<<"\tb="<<b<<"\tc="<<c<<endl;

    // reference capture
    for(int k=0;k<4;++k)
        cout << [&](int x){ a+=x*x; return b+=x; }(k) << endl;

    cout << "a="<<a<<"\tb="<<b<<"\tc="<<c<<endl;

    for(int v:{2,5,7,10})
        cout << [=]() mutable { return sum += a*v; }() << endl;

    cout << "sum="<<sum<<endl;

    return 0;
}
```

# When Are Values Captured?

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a=1,b=2,c=3;

    auto f = [=](int x){ return a*x*x + b*x + c; };

    for(int k=0;k<4;++k)
        cout<<f(k)<<endl;

    a=11; b=12; c=13;

    for(int k=0;k<4;++k)
        cout<<f(k)<<endl;

    return 0;
}
```

Important rule:

- Values captured by value are **copied when the lambda is defined**, not when executed.

# When Are References Captured?

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a=1,b=2,c=3;

    auto f = [&](int x){ a*=x; b+=x; c=a+b; };

    for(int k=1;k<3;f(k++))
        ;

    cout<<"a="<<a<<"\tb="<<b<<"\tc="<<c<<endl;

    a=11;b=12;c=13;

    for(int k=1;k<3;f(k++))
        ;

    cout<<"a="<<a<<"\tb="<<b<<"\tc="<<c<<endl;

    return 0;
}
```

Reference capture means the lambda **directly modifies original variables**.

# Capture by Value vs Reference

Capture-by-value:

- similar to **pass-by-value**
- captured variables cannot be modified unless `mutable` is used.

Example:

```cpp
int a=1,b=2;

cout << [a](int x){ return a+=x; }(20); // error
cout << [b](int x) mutable { return b*=x; }(20); // OK
```

# Example: Mutable Lambda

```cpp
#include <iostream>
using namespace std;

int main()
{
    float a=1.6,b=2.7,c=3.8;

    auto f = [&,a](int x) mutable -> int
    {
        a*=x;
        b+=x;
        return c = a+b;
    };

    for(int k=1;k<3;++k)
        cout<<"f("<<k<<")="<<f(k)<<endl;

    return 0;
}
```

Capture syntax:

```
[&, a]
```

Meaning:

- capture all variables **by reference**
- except `a`, captured **by value**

# Part II  
## Function Pointers

Example:

```cpp
int square(int x){ return x*x; }

int main()
{
    int (*fp)(int) = square;
}
```

`fp` is a pointer to a function returning `int`.

---

# Function Pointer Basics

Functions can be **passed as arguments**.

Example template function:

```cpp
template <typename T>
inline const T& larger(const T& a, const T& b)
{
    return (a<b)?b:a;
}
```

Function pointer type:

```
inline const T& (*)(const T&, const T&)
```

---

# Example: Choosing Function Dynamically

```cpp
#include <iostream>
using namespace std;

int larger(int x,int y){ return (x>y)?x:y; }
int smaller(int x,int y){ return (x>y)?y:x; }

int main()
{
    int choice;
    cin>>choice;

    int (*f)(int,int) = (choice==1)? larger : smaller;

    cout<<f(3,5)<<endl;

    return 0;
}
```

---

# Array of Function Pointers (Calculator)

```cpp
#include <iostream>
using namespace std;

double add(double x,double y){ return x+y; }
double subtract(double x,double y){ return x-y; }
double multiply(double x,double y){ return x*y; }
double divide(double x,double y){ return x/y; }

int main()
{
    double (*f[])(double,double) = {add,subtract,multiply,divide};

    int op;
    double x,y;

    while(cin>>op>>x>>y)
    {
        if(op>=0 && op<=3)
            cout<<f[op](x,y)<<endl;
    }

    return 0;
}
```

# Function Pointer Pointing to Lambda

```cpp
#include <iostream>
using namespace std;

int main()
{
    int choice;
    cin>>choice;

    int (*f)(int,int);

    if(choice==1)
        f = [](int x,int y){ return (x>y)?x:y; };
    else
        f = [](int x,int y){ return (x>y)?y:x; };

    cout<<f(3,5)<<endl;

    return 0;
}
```

Important rule:

Function pointer **cannot point to lambdas with captures**.

# Nested Lambda Example

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a=1,b=1,c=1;

    auto f = [a,&b,&c]() mutable
    {
        auto g = [a,b,&c]() mutable
        {
            cout<<a<<b<<c<<endl;
            a=b=c=4;
        };

        a=b=c=3;
        g();
    };

    a=b=c=2;
    f();

    cout<<a<<b<<c<<endl;

    return 0;
}
```

This example demonstrates **nested lambda scopes and capture rules**.
